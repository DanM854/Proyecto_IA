Librerias Usadas

matplotlib

pyplot

Librería de Python especializada en la creación de gráficos en bidimensiones. Permite crear los tipos de gráficos más comunes.

sklearn

dataset

Genera el conjunto de datos que se utilizara en el modelo, para este caso 1797 muestras de imágenes con dígitos escritos a mano en un tamaño de (8x8) pixeles

svm

Support vector machine, libreria que contiene el conjunto de algoritmos similares a la regrecion linial pero con el objetivo de clacificar los datos.

metrics

Nos permite ver la informacion referente a las metricas de rendimiento del modelo, permitiendo evaluar aspectos como la precision.

train_test_split

Libreria que nos premite separar los datos que van a entrar a la red neuronal entre datos de prueba y de entrenamiento.
