Análisis red neuronal

Librerías Usadas

matplotlib

pyplot

Librería de Python especializada en la creación de gráficos bidimensionales. Permite crear los tipos de gráficos más comunes.

sklearn

dataset

Genera el conjunto de datos que se utilizara en el modelo, para este caso 1797 muestras de imágenes con dígitos escritos a mano en un tamaño de (8x8) pixeles.

svm

Support vector machine, librería que contiene el conjunto de algoritmos similares a la regresión lineal, pero con el objetivo de clasificar los datos.

metrics

Nos permite ver la información referente a las métricas de rendimiento del modelo, permitiendo evaluar aspectos como la precisión.

train_test_split

Librería que nos permite separar los datos que van a entrar a la red neuronal entre datos de prueba y de entrenamiento.

Datos utilizados

Partimos de imágenes de 8x8 pixeles, las cuales guardan los valores de la escala de grises de los pixeles en un arreglo de 8x8 correspondientes a cada muestra.

Transformación

Posterior mente transformamos los datos para que puedan ser procesados por la red neuronal. Primero convertimos el arreglo de 8x8 a una lista de 64 pixeles. Después le damos forma al conjunto de datos en un arreglo que sea (n_imagen,n_pixel).

División

Para que la red funcione necesita ser entrenada y probada, por ello el conjunto de datos en dos uno de pruebas y otro de entrenamiento. Por ello dividimos los datos en 50% para ambos y de la forma (X_train, y_train) y (X_test, y_test).
