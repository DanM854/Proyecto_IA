INTRODUCCIÓN AL PROYECTO

Aqui podras encontrar un proyecto basado en Machine Learning, donde usamos 3 modelos para identificar diferentes cosas, como el reconocimiento de objetos, poses e incluso sonidos, todo esto basandonos en el aprendizaje de los modelos, los cuales al ser entrenados reducen su margen de error al minimo, ademas realizaremos una analisis del funcionamiento de una red neuronal.

MODELOS DE CLASIFICACIÓN

  • Reconocimiento De Imagenes

    Este programa permite el reconocimiento de distintas categorias de imagenes, en este caso identifica un rostro humano con tapabocas, una mano y un celular.

  • Reconocimiento De Sonidos

    El programa de reconocimiento de voz permite identificar distintos sonidos, en este caso el programa reconoce los ladridos de un perro, los grasnidos de un pato, el maullido de un gato y el sonido ambiental de una clase.

  • Reconocimiento De Poses

    El programa de reconocimiento de poses sigue el movimiento humano para poder identificar alguna pose que se este realizando, las tres poses son estiramientos basicos para la movilidad, en la pagina se muestra que poses son las que reconoce.
